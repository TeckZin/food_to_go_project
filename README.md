# Food To Go

Live site: https://www.foodtogo.site/

Food To Go is a web application for college students to browse and order pre-made meals. This project is built with **Vue 3**, **TypeScript**, and **Vite**.

## Tech Stack
- Vue 3 (Single File Components + `<script setup>`)
- TypeScript
- Vite
- Firebase Hosting (deployment)

## Getting Started

### Prerequisites
- Node.js (LTS recommended)
- npm (or pnpm/yarn)
- Firebase CLI (for deployment)

Install Firebase CLI (if needed):
```bash
npm install -g firebase-tools
```

### Install
```bash
npm install
```

### Run locally (development)
```bash
npm run dev
```
Open the local URL shown in your terminal (commonly `http://localhost:5173`).

### Build for production
```bash
npm run build
```

### Preview the production build
```bash
npm run preview
```

## Firebase Hosting (Deploy)

### 1) Authenticate
```bash
firebase login
```

### 2) Deploy Hosting
Run a production build first:
```bash
npm run build
```

Then deploy **only Hosting**:
```bash
firebase deploy --only hosting
```

> Note: The `--only hosting` flag ensures Firebase deploys Hosting only (and does not attempt to deploy other Firebase services).

## Project Structure (Typical)
- `src/` — application source code
- `public/` — static assets copied as-is
- `dist/` — production build output (generated by Vite)
